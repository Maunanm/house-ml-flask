Built with Machine Learning You Can Trust

Our prediction engine uses regression-based machine learning models trained on thousands of property data points. It learns from real housing transactions, listings, and pricing trends to provide predictions you can rely on—updated regularly with the latest data.

Column Dataset

The dataset used in this model consists of 1010 house data in the Tebet and South Jakarta areas with information:

1

LB (Building Area)

The total built-up area in square meters.

2

LT (Land Area)

The total land area in square meters.

3

KT (Bedrooms)

The number of bedrooms.

4

KM (Bathrooms)

The number of bathrooms.

5

GRS (Garage)

The number of garage spaces.

6

House Price

The price of the house

Model Overview

1

price_model.pkl

Function : House price prediction.

Type : Random Forest Regressor

Input : 5 features (LB, LT, KT, KM, GRS)

Output : House price (IDR)

Example Output : Rp 4,500,000,000

2

worth_it_model.pkl

Function : Classification of "Worth it" or "Not Worth it"

Type : Random Forest Classifier

Input : 5 features (LB, LT, KT, KM, GRS)

Output : 1 = Worth it, 0 = Not Worth it

Example Output : 1 (Worth it)

3

feature_scaler.pkl

Function : Normalize input features

Type : StandardScaler

Input : Equalize feature scale (mean=0, std=1)

Output : LB = 200 → 0.5

4

worth_mapping.pkl

Function : Mapping classification label

Type : {'Worth it': 1, 'Not Worth it': 0}

Input : Convert number ↔ label

Output : 1 → "Worth it"

Workflow in Flask App

1

User Input

Enter feature values (example: LB=200, LT=200, KT=3, KM=3, GRS=2)

2

Feature Normalizatiom

Features are normalized using `feature_scaler.pkl`

3

Price Prediction

Normalized results are predicted by `price_model.pkl`

4

Predict Purchase Worthiness

The same data is processed by `worth_it_model.pkl` for classification

5

Convert Classification Labels

0/1 output is converted to text labels using `worth_mapping.pkl`
